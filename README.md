# HoMieS-FooDS-
Este es el código donde se va a poner en funcionamiento todo lo que se necesita para que una aplicación de sugerencia de comidas necesita.
Primero se pone a funcionar el codigo que nos va a ayudar a que las imagenes ya tomadas sean dirigidas a una base de datos donde se separen por 2 actegorias, que en este caso son: Fruver; Mercado. Luego se podran adjuntar mas imagenes para que el mismo codigo nos ayude a acomodarlas en su lugar correspondiente.
